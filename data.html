Running Descriptive Statistics

Analyzing descriptive statistics may lead you to ask new questions. For example, perc_enjoy is much higher on average than perc_usefulness, which also has a large variance. This suggests that a segment of users may not have considered the system useful even though they enjoyed using it.

Steps to Run Descriptive Statistics

  1. Ensure data requiring summary, those with continuous values, are adjacent.
  2. Write down the input range (e.g., H2:J26), ensuring variable labels are not included.
  3. Go to Tools > Add-Ins... and enable Analysis ToolPak.
  4. Go to Tools > Data Analysis and select Descriptive Statistics.
  5. Enter the input range and check Summary Statistics.
  6. Click OK, and the summary will output in a new sheet.
  7. Rename the new sheet to Descriptive_Stats and label the output accordingly.

Running Inferential Statistics in PASW

To run inferential statistics, open PASW and upload the spreadsheet:

  • Go to Open an existing data source > More Files... and select the Formatted_Data worksheet.
  • Use the Analyze menu to find statistical methods such as Wilcoxon-Mann-Whitney test, Simple Linear Regression, and Non-Parametric Correlation.
  • Use the Kruskal-Wallis one-way analysis to determine differences in perc_usefulness between free_play and other tutorial types.
  • Save statistical analysis output by exporting it to Word, Excel, or PDF.
