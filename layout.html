How to Organize and Analyze Research Data

This short lesson is intended for graduate and advanced undergraduate students undertaking original research projects. After completing the 30-minute lesson, you will be able to:

  • Format raw research data in an Excel spreadsheet for efficient analysis
  • Run basic descriptive statistics in Excel
  • Prepare the document for statistical analysis in the software package PASW

Lesson Requirements

For this lesson, you'll need the following:

  • Microsoft Excel
  • PASW (formerly SPSS) statistics software, or free trial download

You should already be familiar with:

  • The basic functions and tools of Excel
  • Descriptive statistics and inferential statistics
  • Research design and how to identify independent variables and dependent variables

Format Data for Analysis

Ensure that Microsoft Excel is installed on your computer. Open the document ToolBox_DataLesson.xls in Excel and follow along with the lesson.

Steps to Format Data:

  1. Each variable name should be unique and start with a letter. Rename 4tutorial_types to tutorial_types.
  2. Ensure names are intuitive: rename use and useful to perc_ease_of_use and perc_usefulness.
  3. Reformat categorical data values for consistency (e.g., change "female" to "F").
  4. Separate data into component values where necessary (e.g., tasks_completed â†’ task1, task2, task3).
  5. Replace categorical data values with "0" or "1" for dichotomous representation.
  6. Copy the Raw_Dataset into Sheet2 and rename it Formatted_Data.
  7. Add new columns for categorical values and replace values using Edit > Replace.

Why Use Dichotomous Variables?

Dichotomous variables allow for easier isolation of characteristics for statistical analysis. For instance, separating experience levels lets researchers test if users with low experience have significantly different perceptions of ease of use compared to medium and high experience levels.
